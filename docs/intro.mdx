## ✍️ 如何新增与管理文档

本知识库基于 **Docusaurus** 与 **MDX** 驱动。你可以非常轻松地添加新内容：

### 新建文档步骤

1. **创建 Markdown/MDX 文件**：
   在 `docs/` 目录下（或其子目录下）新建一个 `.md` 或 `.mdx` 文件。例如：
   ```bash
   touch docs/dev/my-new-tech.md
   ```

2. **添加 Front Matter（页头元数据）**：
   在文件最顶部添加 YAML 格式的配置项，用于控制标题、排序等：
   ```markdown
   ---
   id: my-new-tech
   title: 我的新技术记录
   sidebar_position: 2
   ---
   ```

3. **编写内容**：
   你可以使用标准 Markdown，也可以使用 MDX 特有的 React 组件或高级样式。

### MDX 进阶特性示例

#### 1. 交互式提示框 (Admonitions)
可以使用 `:::` 来包裹提示框，Docusaurus 会自动渲染为精美的样式：

```markdown
:::info 提示
这是一个普通提示框，用于补充背景说明。
:::

:::danger 警告
这是一个高风险操作提示，例如：请勿在生产环境直接运行危险命令。
:::
```

#### 2. 代码高亮与交互
支持主流编程语言的语法高亮，并带有便捷的“复制”按钮：

```javascript
// 示例：快速启动 Express 服务的代码片段
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(PORT, () => {
  console.log(`Server is running on port ${PORT}`);
});
```
